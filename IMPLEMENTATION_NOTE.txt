HOW IT WORKS
================================================================================

PASSWORD STORAGE:
- Location: Browser localStorage
- Key: "admin_credentials"
- Format: { username: "admin", password: "newpassword" }
- Persistence: Survives browser restart and shutdown
- Security: Works on HTTPS (Hostinger provides free SSL)

SESSION MANAGEMENT:
- Location: Browser sessionStorage
- Key: "admin_session"
- Value: "authenticated"
- Persistence: Clears when browser closes
- Behavior: Password change doesn't require logout

NO DATABASE:
✓ No database tables needed
✓ No server-side storage
✓ No migrations required
✓ No database configuration
✓ Works on any hosting (Hostinger included)

NO ENVIRONMENT VARIABLES (Required):
✓ Works immediately out of the box
✓ No setup needed for password features
✓ Email is completely optional
✓ If SMTP not configured, demo mode works anyway

================================================================================
FEATURE 1: CHANGE PASSWORD
================================================================================

Location: Admin dashboard header → "Change Password" button
Function: Change your password anytime you're logged in

HOW TO USE:
1. Login to admin panel
2. Click "Change Password" button (with lock icon, top right)
3. Enter current password
4. Enter new password (minimum 6 characters)
5. Confirm new password
6. Click "Change Password"
7. Success! Use new password to login next time

================================================================================
FEATURE 2: FORGOT PASSWORD
================================================================================

Location: Admin login page → "Forgot your password?" link
Function: Reset password if you forget it

HOW TO USE:
1. Go to /admin/forgot-password
2. Enter your email: admin@amalsahari.com
3. Click "Send Reset Email"
4. Enter new password (minimum 6 characters)
5. Confirm new password
6. Click "Reset Password"
7. Success! Use new password to login

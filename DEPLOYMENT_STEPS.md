# خطوات الرفع على Hostinger - الحل النهائي

## الخطوات السريعة:

### 1️⃣ حمّل الملف من v0
- انقر "Download" في v0
- انتظر تحميل الملف المضغوط (.zip)

### 2️⃣ ارفع على Hostinger
```
Hostinger Dashboard 
  → اسم الموقع: amaalsahari.com
  → عمليات النشر
  → رفع ملفات التطبيق
  → اختر ملف ZIP الجديد
```

### 3️⃣ استخرج الملفات
- الهوستينج سيستخرج تلقائياً
- أو انقر "Extract" في الـ file manager

### 4️⃣ أعد النشر
```
Hostinger Dashboard
  → انقر "Republish" أو "Rebuild"
  → انتظر 2-3 دقائق
```

### 5️⃣ Done! ✅
- الموقع يعمل الآن مع الحل الصحيح
- Admin يعدّل → جميع المستخدمين يرون

## Troubleshooting:

### ❌ Mobile يرى صورة قديمة؟
**الحل:** اضغط Refresh أو مسح Cache
```
Chrome: Ctrl+Shift+R (أو Cmd+Shift+R على Mac)
Safari: تحديث ← اضغط لمدة 3 ثوان
```

### ❌ Desktop يعطي error؟
**الحل:**
1. Hard refresh: `Ctrl+Shift+R`
2. مسح Cache من DevTools
3. جرّب متصفح مختلف (Chrome/Firefox/Safari)

### ❌ الصور لا تظهر؟
**التحقق:**
1. هل تم رفع الصور بشكل صحيح؟
2. هل URL الصور صحيح؟
3. جرّب hard refresh

### ❌ Hostinger يعطي error؟
**الحل:**
- انقر Rebuild (إعادة البناء)
- الانتظار 5 دقائق
- جرّب refresh الصفحة
